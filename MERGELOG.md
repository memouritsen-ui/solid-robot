```markdown

- Added: [count]
- Passing: [count]
- Coverage: [%]

**Validation**:
- [ ] Checklist complete
- [ ] All tests pass
- [ ] No lint errors
- [ ] No type errors

**Known Issues**: [Any issues to address later]

**Merge Commit**: [hash]
```
